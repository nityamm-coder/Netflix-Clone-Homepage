# Netflix Clone

A responsive Netflix India landing page clone built with HTML and CSS. This project replicates the look and feel of Netflix's homepage, featuring a hero section, feature highlights, FAQ accordion, and footer.



## Features

- **Hero Section** – Full-width hero with Netflix branding, headline, and email signup CTA
- **Feature Sections** – Showcases key Netflix features:
  - Watch on TV (smart TVs, PlayStation, Xbox, etc.)
  - Download shows for offline viewing
  - Watch everywhere (phone, tablet, laptop, TV)
  - Create profiles for kids
- **FAQ Section** – Frequently asked questions with expandable-style accordion boxes
- **Footer** – Links for Help Centre, Account, Terms of Use, and more
- **Responsive Design** – Optimized for desktop and mobile screens

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid)
- Google Fonts (Poppins, Martel Sans)



### Prerequisites

No build tools required. Just a modern web browser.
